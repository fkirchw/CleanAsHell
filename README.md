# Clean as Hell

*A fast-paced 2D platformer about revenge, blood, and cleaning up Hell.*

## About the Game
**Clean as Hell** is a Unity-based platformer where you play as a **cleaning demon** seeking revenge on Lucifer himself.  
Fight through the depths of Hell, defeat enemies and bosses, and clean up the bloody mess they leave behind.

## Features
- Fast-paced 2D platforming combat
- Unique **cleaning & blood mechanics**
- Multiple enemy types and boss fights
- Traps, shortcuts, and hidden paths
- Upgrade system to enhance player abilities

## Controls
- **Move:** A / D / S  
- **Jump:** W / Space  
- **Clean:** Shift  
- **Upgrade Screen:** E  
- **Pause / Settings:** Esc
